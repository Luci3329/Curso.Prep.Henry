HOMEWORK 1 Archivo de Texto VARIABLE Es una entidad a la que le podemos asignar un valor cualquiera, incluso ese valor puede ser cambiado o reemplazado por otros valores. Se diferencia de una CONSTANTE por que ésta mantiene el valor que le asignamos sin cambios ni modificaciones durante la ejecución del programa. STRINGS Se trata de un TIPO DE DATO. Los tipos de datos pueden ser: NUMÉRICOS (Enteros o Reales) CARACTERES (1 caracter: char o una cadena: STRING) LÓGICOS (Booleanos: verdadero - falso) FUNCIONES Son tipos particulares de objetos que agrupan códigos que luego podemos "invocar" y ejecutar. Las funciones necesitan parámetros ó ARGUMENTOS, que son variables que declaramos dentro de la función y que van a tomar un valor cuando la invoquemos. RETURN Es una sentencia de java script que finaliza la ejecucion de una función y devuelve un valor. DECLARACIÓN "IF" Forma parte de una de las estructuras de CONTROL DE FLUJOS DE datos. Sirve para evaluar una o varias expresiones lógicas y ejecutar instrucciones en caso de que esa expresión se evalúe verdadera. La declaración IF puede ser simple, compuesta o anidada, dependiendo de la cantidad de expresiones lógicas que tenga para evaluar y las instrucciones que tengamos a ejecutar. VALORES BOOLEANOS Son TIPOS DE DATOS lógicos. Solo hay 2 valos booleanos: Verdadero (true) y falso (false). Obtenemos éstas valores como resultado de una comparación.
